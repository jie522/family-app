Google Sheet 同步

全家共用一份 FAMIAP Google Sheet。 啟用後,追劇清單和股票追蹤清單都會自動存進 Sheet(不再只存在這支手機裡),App 裡的新增、記錄、評分、筆記都會即時寫回,家人打開 App 也會看到同一份清單。
啟用方式:照 apps-script/Code.gs 檔案開頭的 5 步驟部署,把網址貼到下面。

TMDB 電影資料庫

申請免費的 API 金鑰後,新增劇時就會自動抓海報和簡介。
申請方式:到 themoviedb.org 註冊 → 設定 → API → 申請 API Key(v3)。
這支手機可以自己貼金鑰(下面),或者只設定一次就好:把金鑰存進 Apps Script 的「指令碼屬性」 (見 apps-script/Code.gs 檔案開頭選用步驟),啟用 Google Sheet 同步後全家都會自動有海報,不用每支手機各貼一次,金鑰也不會出現在公開的程式碼裡。

資料備份

資料存在這支手機的瀏覽器裡。換手機或想跟家人共用清單時,可以匯出檔案再到另一支手機匯入。
