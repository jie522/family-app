# 晶圓代工

晶片產業的分工模式:**設計晶片的公司**(輝達、蘋果、聯發科——只畫圖不蓋廠)把設計圖交給**製造晶片的代工廠**,在矽晶圓上把電路做出來。台積電 1987 年開創此模式,讓設計公司不必花數百億美元蓋廠。

## 產業結構重點

- **先進製程是入場券**:電晶體越小(7nm→5nm→3nm→2nm)晶片越快越省電;全球能量產最先進製程的只剩台積電、三星、英特爾三家
- **贏者全拿的正循環**:先進製程一座廠 200~300 億美元 → 只有接得到大單的敢投資 → 技術領先 → 更大單 → 更有錢投資
- **先進封裝(CoWoS)**:AI 晶片要把 GPU 和高頻寬記憶體(HBM)疊在一起[封裝](k:packaging-testing),產能長期供不應求,是 AI 晶片出貨瓶頸
- 產業鏈:上游設備(ASML 的 EUV)/矽晶圓(環球晶)→ 中游製造 → 下游設計、封測(日月光)、組裝(鴻海)

## 為什麼重要

台灣以台積電為核心的完整聚落,讓全球最尖端晶片高度集中在台灣生產——既是護城河,也是地緣政治焦點。是理解[AI供應鏈](k:ai-supply-chain)的基礎。
